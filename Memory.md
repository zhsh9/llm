## 大模型推理显存

$$
Mem = (ParamNum * 4 * L)*(1+O)
$$

- ParamNum: 模型参数数量（以B|十亿为单位）
- 4*L: 每个参数使用的比特数
- 4: 4字节精度
- L: 量化精度系数
  - 全精度（FP32），L = 1
  - 半精度（FP16），L = 0.5
  - 8位量化（INT8），L = 0.25
  - 4位量化（INT4），L = 0.125
  - 3位量化（INT3），L = 0.09375 (3/32)
  - 2位量化（INT2），L = 0.0625 (1/16)
  - 1位量化（二值化），L = 0.03125 (1/32)
- O: 额外开销，优化器状态、梯度、激活，大概 0.1~0.3
